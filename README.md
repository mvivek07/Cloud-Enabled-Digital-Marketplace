A large amount of fresh farm produce is discarded every year due to failing cosmetic standards set by supermarkets. Farmers suffer financial losses, while many local businesses such as juice bars, caterers, restaurants, and community kitchens need affordable produce where appearance does not matter.
This project addresses this issue by building a digital platform (web & mobile) to connect farmers with local businesses to buy and sell surplus or cosmetically imperfect produce directly.
